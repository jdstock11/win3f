# Institutional Derivatives Analytics

Live Demo: [https://rolloverdata-beta.vercel.app/](https://rolloverdata-beta.vercel.app/)

Advanced Options Rollover, Calendar Spread Engine & Smart Money Positioning platform.

## Features
- **Institutional Terminal:** Analyze detailed Option Chain files for complex derivatives analysis.
- **Market Activity Scanner:** Identify Institutional Activity and Smart Money Bias by parsing NSE Most Active Contracts data (CSV/Excel).
